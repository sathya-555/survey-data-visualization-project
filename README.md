# Survey Data Visualization Project

## Problem Statement
Survey results are often under-utilized due to poor analysis.

## Objective
To analyze survey data and create informative and engaging visual summaries.

## Dataset
Dummy survey dataset containing:
- Multiple-choice responses
- Rating-based feedback

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualizations
- Bar Chart (Learning Mode Preference)
- Pie Chart (Gender Distribution)
- Heatmap (Ratings Correlation)

## Key Takeaways
- Online learning is preferred by most respondents.
- High satisfaction among interns.
- Strong positive relationship between satisfaction and internship rating.

## How to Run
1. Install required libraries
2. Open `survey_analysis.ipynb`
3. Run all cells

## Interactive Dashboard
An interactive dashboard was created using Plotly to visualize:
- Learning mode preferences
- Gender distribution
- Satisfaction ratings
- Correlation between ratings

The dashboard provides hover and zoom functionality for better insights.
